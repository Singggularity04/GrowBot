# GROW BOT — Telegram-бот воронка продаж для мастеров красоты

Бот-продавец, который доводит клиента до записи через воронку.  
Не заменяет систему записи (Dikidi и т.д.), а **увеличивает конверсию**.

## Возможности

- 🎯 Быстрая запись — минимум действий до ссылки на Dikidi
- 🧠 Подбор услуги — 2-3 вопроса → персональная рекомендация
- 📸 Портфолио — примеры работ с CTA
- ❓ FAQ — снятие страхов → запись
- 🔥 Автодожим — 3 follow-up сообщения тем, кто не записался
- 📊 Статистика — конверсия в реальном времени

## Установка

```bash
# 1. Клонировать / скопировать проект
# 2. Установить зависимости
pip install -r requirements.txt

# 3. Скопировать и заполнить .env
cp .env.example .env
# отредактировать .env — вставить BOT_TOKEN, ADMIN_ID, DIKIDI_LINK

# 4. Запустить бота
python bot.py
```

## Настройка

Все параметры в файле `.env`:

| Переменная     | Описание                          |
|----------------|-----------------------------------|
| `BOT_TOKEN`    | Токен от @BotFather               |
| `ADMIN_ID`     | Telegram ID мастера               |
| `DIKIDI_LINK`  | Ссылка на запись (Dikidi и т.д.)  |
| `MASTER_NAME`  | Имя мастера                       |
| `MASTER_CITY`  | Город                             |
